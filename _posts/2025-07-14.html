Vetores e Matrizes em C: 20 Exemplos para Iniciantes

Este documento contém 20 exemplos de vetores e matrizes na linguagem de programação C, com explicações detalhadas para pessoas leigas. Cada exemplo inclui:

  • Código comentado: Explicação linha por linha.
  • Lógica por trás: O que o código faz e por que é útil.
  • Saída esperada: O que aparece na tela ao executar o programa.

Os exemplos progridem de conceitos simples (vetores) a mais complexos (matrizes). Copie os códigos manualmente para fixar o aprendizado e teste-os em um compilador C (como GCC, Dev-C++, ou OnlineGDB).

O que são vetores e matrizes?

  • Vetor: Uma lista de "caixinhas" na memória, cada uma com um valor (ex.: números, como {1, 2, 3}).
  • Matriz: Uma tabela com linhas e colunas, como uma planilha (ex.: {{1, 2}, {3, 4}}).
  • Array: Nome geral para vetores (1 dimensão) e matrizes (2 ou mais dimensões).

Nota: Em C, arrays têm tamanho fixo definido na criação, e usamos a biblioteca stdio.h para entrada/saída.

Exemplo 1: Criando um vetor simples

#include <stdio.h>

int main() {
    // Criando um vetor com 5 números
    int numeros[5] = {1, 2, 3, 4, 5};
    
    // Mostrando o vetor
    printf("Vetor: ");
    for (int i = 0; i < 5; i++) {
        printf("%d ", numeros[i]);
    }
    printf("\n");
    
    return 0;
}

Explicação

  • #include <stdio.h>: Inclui a biblioteca para usar printf (mostrar na tela).
  • int main(): Onde o programa começa.
  • int numeros[5] = {1, 2, 3, 4, 5};: Cria um vetor com 5 números inteiros.
  • for (int i = 0; i < 5; i++): Conta de 0 a 4 (índices do vetor).
  • printf("%d ", numeros[i]): Mostra cada número.
  • printf("\n"): Pula uma linha.
  • return 0: Finaliza o programa.

Lógica

Cria e exibe uma lista de números. Útil para armazenar sequências, como notas ou idades.

Saída

Vetor: 1 2 3 4 5

Exemplo 5: Tamanho de um vetor

Exemplo 9: Criando uma matriz simples

  • printf("%d ", matriz[i][j]): Mostra o elemento na linha i, coluna j.
  • printf("\n"): Pula uma linha após cada linha da matriz.

  • if: Se o número for encontrado, marca encontrado como 1 e sai do laço.

Exemplo 17: Transpondo uma matriz

#include <stdio.h>

int main() {
    // Criando uma matriz 2x3
    int matriz[2][3] = {{1, 2, 3}, {4, 5, 6}};
    int transposta[3][2];
    
    // Calculando a transposta
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 2; j++) {
            transposta[i][j] = matriz[j][i];
        }
    }
    
    // Mostrando a transposta
    printf("Matriz transposta:\n");
    for (int i = 0; i < 3; i++) {
        for (int j = 0; j < 2; j++) {
            printf("%d ", transposta[i][j]);
        }
        printf("\n");
    }
    
    return 0;
}

Explicação

  • int matriz[2][3]: Cria uma matriz 2x3.
  • int transposta[3][2]: Cria uma matriz 3x2 para a transposta.
  • transposta[i][j] = matriz[j][i]: Troca linhas por colunas.
  • for: Mostra a matriz transposta.

Lógica

A transposta troca linhas por colunas. Útil em matemática e ciência de dados.

    Exemplo 1: Criando um vetor simples

    #include <stdio.h>

int main() {
    // Criando um vetor com 5 números
    int numeros[5] = {1, 2, 3, 4, 5};
    
    // Mostrando o vetor
    printf("Vetor: ");
    for (int i = 0; i < 5; i++) {
        printf("%d ", numeros[i]);
    }
    printf("\n");
    
    return 0;
}

    Explicação

    • #include <stdio.h>: Inclui a biblioteca para usar printf (mostrar na tela).
    • int main(): Onde o programa começa.
    • int numeros[5] = {1, 2, 3, 4, 5};: Cria um vetor com 5 números inteiros.
    • for (int i = 0; i < 5; i++): Conta de 0 a 4 (índices do vetor).
    • printf("%d ", numeros[i]): Mostra cada número.
    • printf("\n"): Pula uma linha.
    • return 0: Finaliza o programa.

    Lógica

    Cria e exibe uma lista de números. Útil para armazenar sequências, como notas ou idades.

    Saída

    Vetor: 1 2 3 4 5

    Exemplo 5: Tamanho de um vetor

    #include <stdio.h>

int main() {
    // Criando um vetor
    int cores[4] = {1, 2, 3, 4};
    
    // Calculando o tamanho
    int tamanho = sizeof(cores) / sizeof(cores[0]);

    • tamanho = sizeof(cores) / sizeof(cores[0]): Divide para obter o número de elementos.
    • printf: Mostra o tamanho.

    Conta quantos elementos estão no vetor. Útil para saber o tamanho de uma lista.

    Exemplo 8: Encontrando o maior valor em um vetor

    #include <stdio.h>

    // Criando um vetor
    int numeros[5] = {4, 7, 2, 9, 1};
    
    // Começando com o primeiro número

    • int maior = numeros[0];: Assume que o primeiro número é o maior.

    • printf: Mostra o maior número.

    Exemplo 14: Verificando se um número está no vetor

    #include <stdio.h>

int main() {
    // Criando um vetor
    int numeros[4] = {5, 10, 15, 20};
    
    // Lendo o número a buscar
    int busca;
    printf("Digite um número para buscar: ");
    scanf("%d", &busca);
    
    // Verificando
    int encontrado = 0;
    for (int i = 0; i < 4; i++) {
        if (numeros[i] == busca) {
            encontrado = 1;
            break;
        }
    }
    
    // Mostrando o resultado
    if (encontrado) {
        printf("O número %d está no vetor!\n", busca);
    } else {
        printf("O número %d não está no vetor.\n", busca);
    }
    
    return 0;
}

    Explicação

    • int numeros[4] = {5, 10, 15, 20};: Cria um vetor.
    • scanf: Lê o número a buscar.
    • int encontrado = 0;: Variável para indicar se o número foi encontrado.
    • for: Verifica cada elemento.
    • if: Se o número for encontrado, marca encontrado como 1 e sai do laço.
    • printf: Mostra o resultado.

    Lógica

    Verifica se um número está no vetor. Útil para buscas, como procurar um aluno em uma lista.

    Saída (exemplo com entrada 10)
